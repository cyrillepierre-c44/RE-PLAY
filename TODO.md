# Home Page Redesign Plan

## Task
Redesign the home page (`app/views/pages/home.html.erb`) with similar card layout as sign_up and log_in pages but with different colors.

## Steps
1. [x] Read and understand the sign_up, log_in, and current home page designs
2. [x] Create the new home page design with blue/aqua theme
3. [x] Test the implementation

## Design Specs (Blue/Aqua Theme)
- **Background gradient**: `#4FACFE` to `#00F2FE` (fresh blue)
- **Card**: white with backdrop-filter blur, rounded corners (24px)
- **Header illustration**: gradient `#06B6D4` to `#0891B2` (cyan)
- **Primary buttons**: blue/cyan gradient matching header
- **Floating shapes**: blue tones with float animations
- **Typography**: Poppins font family
- **Responsive**: mobile-friendly with proper breakpoints

## Design Elements to Include
1. Animated floating background shapes
2. Modern card with illustration section
3. Welcome message and tagline
4. Action buttons for login/signup
5. Guest access option with styled links
